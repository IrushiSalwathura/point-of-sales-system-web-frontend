# POINT OF SALES SYSTEM (FRONTEND)

A simple point of sales application user interfaces that are built with following technologies.
* HTML
* CSS and SCSS
* ECMAScript 5
* Bootstrap
* jQuery
* AdminLTE

## Usage

1. Clone the repository

`git clone https://github.com/IrushiSalwathura/point-of-sales-system-web-frontend.git`

2. Open it in IntelliJ IDEA OR a preferred IDE

3. Install all the dependencies

`npm install`

4. Set all the file watchers

5. You are all set.
